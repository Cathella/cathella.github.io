Homeschool - (Current Project)

Homeschool is a project that will avail students with practice questions in different study topics in the form of quizzes, lessons and discussions. Teachers will also be able to sign up to create or upload lessons for their students.

Features
  • An authentication system for students and teachers
  • Students can view and comment on lessons
  • Teachers can upload lessons
  • Teachers can submit questions and answers for a Quiz question
  • Quiz questions should verified by the Admin
  • Both Teachers and Students can create a discussion
  • Both Teachers and Students can participate in the discussions through thread comments.
Technologies
ReactJS + Redux (Frontend), Ruby on Rails API (Backend).
